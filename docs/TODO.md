(`EconomyTick.ts`), добыча ресурсов считается только в `ResourceTick`. Без живой

- **РЕШЕНО (2026-06-23), готово к коду:** `taxRevenue = gdp × taxRate` (ставка выводится в
  `createGame`); добыча — только `ResourceTick` (вопрос 4 закрыт); export/stateEnterprise/other —
  статичны в первом проходе. Спецификация и точки правки — в `DECISIONS.md`.
- Реализация: тип `EconomyState` (поле `taxRate`) + `createGame` + `EconomyTick` + обновить тест.
